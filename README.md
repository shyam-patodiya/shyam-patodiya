## 🛠️ Solutions & Integrations

Beyond standard frameworks, we build robust, real-world systems:

- **Enterprise Dashboards** — Angular apps with ag-Grid, Gantt charts, ApexCharts, ECharts, Mapbox, vis-timeline, PDF export pipelines, role-based authentication, and bilingual EN/AR RTL support.
- **Project Management Systems** — Custom PM tools built on AppSheet + Google Sheets + Apps Script; handling engineering investigation workflows across structural, fire/smoke, water damage, and wind/hail categories. Includes role-based access, virtual columns, archiving logic, financial summaries, and automated notification engines.
- **No-Code / Low-Code Platforms** — Advanced AppSheet configurations with complex formulas, enum management, date formatting, and slice-based views for different stakeholder roles.
- **AI-Assisted Development** — Integrating Claude Code, Cursor AI, and MCP servers into daily workflows. Custom CLAUDE.md configurations and project SOPs for consistent, accelerated delivery.
- **PDF & Document Automation** — Payload generation pipelines for complex nested form structures; programmatic document creation and export.
